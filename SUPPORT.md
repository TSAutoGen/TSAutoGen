# Support

## How to get help and find support

TSAutoGen is a community-driven project. We use several channels to help you get the most out of the framework.

### GitHub Issues

Please use [GitHub Issues](https://github.com/TSAutoGen/TSAutoGen/issues) to:
*   Report bugs.
*   Request new features.
*   Track progress on existing issues.

**Before filing a new issue:**
1.  Search existing issues to avoid duplicates.
2.  Provide a clear description of the problem or request.
3.  For bugs, include steps to reproduce and system info.

### GitHub Discussions

For general help, questions, and community discussions, please use [GitHub Discussions](https://github.com/TSAutoGen/TSAutoGen/discussions).

### X/Twitter

Follow us and reach out to us on X/Twitter: [@TSAutoGen](https://x.com/TSAutoGen).
