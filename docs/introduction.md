# Introduction to TSAutoGen

TSAutoGen is a high-performance, TypeScript-native port of Microsoft's AutoGen framework. It enables developers to build next-generation multi-agent AI systems that can converse, collaborate, and execute tasks autonomously.

## Why TSAutoGen?

While the original AutoGen is powerful, the TypeScript ecosystem offers unique advantages for modern web and server-side development:

- **Type Safety**: Leverage TypeScript's robust type system to build more reliable agent interactions.
- **Node.js Performance**: Optimized for asynchronous workflows and non-blocking I/O.
- **Unified Ecosystem**: Seamlessly integrate your AI agents with React, Next.js, and other modern web frameworks.
- **Native ESM**: Built for the future with modern JavaScript standards.

## Core Capabilities

TSAutoGen focuses on three main pillars:
1. **Conversable Agents**: Agents that can receive, process, and send messages with customizable roles.
2. **Multi-Agent Orchestration**: Managing complex interactions through patterns like Group Chat.
3. **Autonomous Execution**: Enabling agents to write and run code to solve problems in real-time.

## Support for Multiple LLMs

TSAutoGen provides a unified interface for the most powerful models today:
- **OpenAI**: GPT-4, GPT-4o, and more.
- **Google**: Gemini 1.5 Flash and Pro.
- **Anthropic**: Claude 3.5 Sonnet and Opus.
