# FAQ (Frequently Asked Questions)

## General Questions

### What is TSAutoGen?
TSAutoGen is a TypeScript-native port of Microsoft's AutoGen. It's designed for developers building multi-agent AI systems in a Node.js environment.

### Why use TypeScript instead of Python for AutoGen?
TypeScript offers excellent type safety, performance, and a massive ecosystem of web and server-side tools. It's ideal for developers already working with Node.js, Next.js, or other web-native technologies.

### Is TSAutoGen official?
TSAutoGen is a community-driven port inspired by the architecture and concepts of the original Microsoft AutoGen. It aims for high architectural parity while being optimized for TypeScript.

## Technical Questions

### How do I install TSAutoGen?
Simply use npm: `npm install tsautogen`.

### Which LLMs are supported?
Currently, we support:
*   **OpenAI** (all models including GPT-4o)
*   **Google Gemini** (gemini-1.5-flash, etc.)
*   **Anthropic Claude** (claude-3-5-sonnet, etc.)

### Can I run my own code locally?
Yes, TSAutoGen includes a `LocalCodeExecutor` that can execute both Python and Node.js code generated by agents.

### Does TSAutoGen support asynchronous execution?
Yes, the entire framework is built with `async/await` and is optimized for the non-blocking nature of Node.js.
