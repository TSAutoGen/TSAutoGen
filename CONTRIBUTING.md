# Contributing to TSAutoGen

We are excited that you're interested in contributing to TSAutoGen! The goal of this project is to foster a collaborative and inclusive community where diverse perspectives can drive innovation in the multi-agent AI space.

## How to Contribute

### 1. Pushing Patches
If you have a bug fix or improvement, please:
1.  Fork the repository.
2.  Create a feature branch.
3.  Write tests for your changes.
4.  Submit a Pull Request.

### 2. Code Review
We value peer review! Reviewing PRs from others is a great way to contribute and learn about the codebase.

### 3. Documentation and Examples
Improving the README, adding new examples in `examples/`, or writing tutorials is highly appreciated.

### 4. Community Participation
Participate in [GitHub Issues](https://github.com/TSAutoGen/TSAutoGen/issues) and [Discussions](https://github.com/TSAutoGen/TSAutoGen/discussions).

## Code Style

TSAutoGen uses TypeScript and follows modern ESM standards.
*   Always use `await` with async functions.
*   Enforce type safety with clear interfaces.
*   Write descriptive commit messages.

## Code of Conduct

All contributors are expected to follow our [Code of Conduct](CODE_OF_CONDUCT.md).

## Setting Up Locally

1.  Clone the fork: `git clone https://github.com/YOUR_USER/TSAutoGen.git`
2.  Install dependencies: `npm install`
3.  Run tests: `npm test`
4.  Run a local example: `node --loader ts-node/esm examples/basic_chat.ts`
